**Sowdhanya Laxna – Developer Portfolio**

A modern, responsive, and animation-rich personal portfolio built using React.js. This portfolio showcases my projects, skills, experience, and creative UI/UX capabilities with smooth animations and interactive components.

**Features**
Modern UI/UX design
Smooth animations and transitions
Fully responsive (Mobile + Desktop)
Fast and optimized performance
Component-based architecture
Interactive sections for projects, skills, and experience

**Tech Stack**
Frontend: React.js
Styling: CSS / Tailwind (if used)
Animations: Custom animations & reusable components
Version Control: Git & GitHub


Follow these steps to run the project locally:

# Clone the repository
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# Navigate to project folder
cd YOUR_REPO_NAME

# Install dependencies
npm install

# Start development server
npm run dev
